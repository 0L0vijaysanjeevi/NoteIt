# NoteIt

**NoteIt** is a user-friendly browser extension that allows you to save and organize all your notes and links effortlessly.

---

## Features

- **Text Input**: Easily enter plain text notes.
- **Link Saving**: Store URLs for quick access.
- **Simple Display**: View all saved notes and links in one area.

---

## Tech Stack

- **HTML, CSS, and JavaScript**: For a clean and modern user interface.

---

## Installation

1. Download or clone the repository:
   ```bash
   git clone https://github.com/0L0vijaysanjeevi/NoteIt.git
   ```
   
2. Open `index.html` in your web browser to start using NoteIt.

---

## Files

- **`index.html`**: Structure and markup.
- **`index.css`**: Styling.
- **`index.js`**: Logic for saving and displaying notes and links.
